APIs

  • Binding - A modern databinding library for JavaScript and HTML.
  • Bootstrapper - Sets up the default configuration for the aurelia framework and gets you up and running quick and easy.
  • Dependency Injection - A lightweight, extensible dependency injection container for JavaScript.
  • Event Aggregator - A lightweight pub/sub messaging system for app-wide or per-object loosely coupled events.
  • Fetch - A simple client based on the Fetch standard.
  • Framework - The aurelia framework brings together all the required core aurelia libraries into a ready-to-go application-building platform.
  • History - An abstract module which specifies the interface for history implementations used by a router.
  • History-Browser - An implementation of the Aurelia history interface based on standard browser hash change and push state mechanisms.
  • I18N - A plugin that provides i18n support.
  • Loader - An abstract module which specifies an interface for loading modules and view templates.
  • Loader-Default - A default implementation of the loader interface compatible with system.js and require-based loaders.
  • Logging - A minimal but effective logging mechanism with support for log levels and pluggable log appenders.
  • Logging-Console - A console log appender for the Aurelia logging library.
  • Metadata - Utilities for reading and writing the metadata of JavaScript functions.
  • Platform Abstraction Layer (PAL) - Aurelia's platform abstraction layer (PAL).
  • PAL-Browser - The browser-specific implementation of Aurelia's platform abstraction layer.
  • Path - Utilities for path manipulation.
  • Router - A powerful client-side router.
  • Task Queue - A simple task queue for the browser that enables the queuing of both standard tasks and micro tasks.
  • Templating - An extensible HTML templating engine supporting databinding, custom elements, attached behaviors and more.
  • Templating-Binding - An implementation of the templating engine's Binding Language abstraction which uses a pluggable command syntax.
  • Templating-Resources - A standard set of behaviors, converters and other resources for use with the Aurelia templating library.
  • Templating-Router - An implementation of the RouteLoader interface for use with the router module. Also contains a custom element that allows the templating engine to display the current route.
  • Testing - A collection of helpers for testing Aurelia apps and components.
